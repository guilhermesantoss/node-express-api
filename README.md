## Instalando as dependências do projeto

Utilize o comando:
```bash
$ npm install

# OR

$ yarn install
```

Agora que as dependências estão devidamente instaladas, vamos executar utilizando o comando:
```bash
$ npm run dev

# OR

$ yarn dev
```
